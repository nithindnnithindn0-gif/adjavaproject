# Complaint Management System

##  Description

This is a Java Swing-based application used to manage user complaints. It allows users to submit complaints and admins to view and resolve them.

##  Features

* User Login System
* Submit Complaint
* View Complaints
* Admin Dashboard to update status

##  Technologies Used

* Java (Swing)
* MySQL Database
* JDBC

##  How to Run

1. Open project in IntelliJ
2. Configure database connection
3. Run LoginUI.java

##  Project Structure

* db → Database connection
* dao → Data access logic
* ui → User interface
